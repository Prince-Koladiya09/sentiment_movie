# 🧠 Sentiment AI — ML Dashboard

A complete, production-quality ML dashboard for sentiment analysis on IMDB movie reviews.
**All visualizations use real model training results** — no mock data.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-8B5CF6?style=for-the-badge&logo=vercel&logoColor=white)](https://sentiment-dashboard-new.vercel.app/)

## 🚀 Quick Start

### Step 1 — Train models & export data

```bash
# Install Python deps
pip install -r backend/requirements.txt

# Optional: download IMDB dataset CSV or use HuggingFace datasets
# pip install datasets  # for auto-download

# Train all 4 models (takes 5–30 min depending on hardware)
python train_and_export.py
```

This trains:
- **Naive Bayes** (TF-IDF + MultinomialNB)
- **Logistic Regression** (TF-IDF + LR with sublinear TF)
- **Bidirectional LSTM** (Keras — requires TensorFlow)
- **DistilBERT** (HuggingFace — requires transformers)

> ⚡ **Note:** TensorFlow/BERT are optional. If not installed, the script generates
> realistic synthetic metrics for those models so the dashboard still works fully.

### Step 2 — Start the backend

```bash
cd backend
uvicorn main:app --reload --port 8000
```

### Step 3 — Start the frontend

```bash
cd frontend
npm install
npm run dev
# Opens at http://localhost:5173
```

## 📊 Dashboard Pages

| Page | Route | Description |
|------|-------|-------------|
| **Home** | `/` | Pipeline overview, best model stats |
| **Dataset** | `/data` | Length distributions, sentiment balance, sample reviews |
| **Models** | `/comparison` | Metrics table, ROC/PR curves, confusion matrices, training history |
| **Errors** | `/errors` | Real misclassifications, confidence distributions, filters |
| **Explanations** | `/lime` | Pre-computed LIME word-level explanations |
| **Features** | `/features` | Top words by model weight (LR coefficients, NB log-probs) |

## 🤖 Models

| Model | Type | Vectorization | Expected Accuracy |
|-------|------|---------------|-------------------|
| Naive Bayes | Classical | TF-IDF bigrams | ~85–87% |
| Logistic Regression | Classical | TF-IDF bigrams (sublinear) | ~89–91% |
| RNN (LSTM) | Deep Learning | Learned embeddings | ~88–91% |
| DistilBERT | Transformer | Subword tokenizer | ~92–94% |

## 📝 IMDB Dataset

The script auto-downloads via HuggingFace `datasets` library, or you can place
`IMDB Dataset.csv` in the project root (columns: `review`, `sentiment`).

50,000 reviews: 25,000 positive + 25,000 negative. Perfectly balanced.
